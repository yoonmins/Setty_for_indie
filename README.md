## 🎯 프로젝트 소개
**Setty_for_indie** 는 홍익대학교 산업디자인학과 졸업 전시회(2024)를 위해 제작된 웹사이트입니다.  
짧은 제작 기간(2일)에도 불구하고 빠르게 MVP를 구현하였으며, 이후 요구사항 변경을 반영하여 **원페이지 반응형 웹사이트**로 발전시켰습니다.
